☁️ Synchronisation entre téléphones

Utilise JSONBin.io (gratuit) pour partager les données entre vos deux téléphones.

📋 Comment obtenir le Bin ID et la clé API ?
1. Va sur jsonbin.io et crée un compte gratuit
2. Clique sur + New Bin
3. Colle {} et clique Create Bin
4. Copie l'ID du bin (dans l'URL)
5. Va dans API Keys → copie ta Master Key
6. Colle les deux ici et sauvegarde
7. Fais pareil sur le téléphone de ton mari
